# Rochambeau (Rock Paper Scissors) — JavaScript

A Rock Paper Scissors game built with vanilla JavaScript as part of **The Odin Project** Foundations.

The game runs in the browser using `prompt()` for user input and logs results to the **DevTools console**.

---

## Live Preview

* Repo: [https://github.com/Mafresh-BB/Rochambeau](https://github.com/Mafresh-BB/Rochambeau)

---

## Features

* Random computer choice (`rock`, `paper`, `scissors`)
* Validates user input (rejects invalid entries)
* Input normalization (trims spaces + case-insensitive)
* Supports **Cancel to quit**
* Plays **5 rounds**
* Tracks and displays score after each round
* Prints final score at the end

---

## How To Run

1. Clone the repo:

```bash
git clone https://github.com/Mafresh-BB/Rochambeau.git
```

2. Open the project folder:

```bash
cd Rochambeau
```

3. Open `index.html` in your browser.

---

## How To Play

* Enter **rock**, **paper**, or **scissors** when prompted
* Results are shown in the browser console

### Open the Console

* **Windows/Linux:** `Ctrl + Shift + J`
* **Mac:** `Cmd + Option + J`
